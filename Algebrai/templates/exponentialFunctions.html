What is an Exponential Function?

An exponential function is a function where the variable is in the exponent. The general form is:

f(x) = a * b^x

where a is a constant, b is the base (a positive number), and x is the exponent (the variable).

Exponential Growth vs. Exponential Decay

Exponential functions can represent growth or decay, depending on the base b:

  • Exponential Growth: If b > 1, the function represents growth, and the graph rises rapidly as x increases.
  • Exponential Decay: If 0 < b < 1, the function represents decay, and the graph falls rapidly as x increases.

Example (Growth):

f(x) = 2 * (1.5)^x

Example (Decay):

f(x) = 3 * (0.5)^x

Key Properties of Exponential Functions

Exponential functions have several important properties:

  • Y-intercept: The y-intercept occurs at (0, a).
  • Asymptote: The graph approaches the x-axis (y = 0) but never crosses it.
  • Domain: The domain is all real numbers.
  • Range: The range is y > 0 for growth and decay functions where a > 0.

Graphing Exponential Functions

To graph an exponential function, plot several points by choosing values of x and finding the corresponding values of f(x). Key points to plot include the y-intercept at (0, a) and points around x = 0 to observe growth or decay.

Example: Graph the function f(x) = 2 * 3^x

  • f(0) = 2 * 3^0 = 2 (y-intercept)
  • f(1) = 2 * 3^1 = 6
  • f(-1) = 2 * 3^-1 = 2/3

Solving Exponential Equations

To solve exponential equations, you often use logarithms to "bring down" the exponent and solve for x. If the equation has the same base on both sides, you can set the exponents equal to each other.

Example 1:

2^x = 8

Since 8 is the same as 2^3, we can set the exponents equal:

x = 3

Example 2:

3^(2x) = 81

81 is the same as 3^4, so:

2x = 4
x = 2
