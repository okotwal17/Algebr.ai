What is a Linear Equation?

A linear equation is an equation where the highest exponent of the variable (usually x) is 1. It represents a straight line when plotted on a graph.

Basic Example

Let's start with a basic linear equation:

               2x + 3 = 11

To solve this equation, we need to isolate the variable x. Follow these steps:

  1. Subtract 3 from both sides: 2x = 8
  2. Divide both sides by 2: x = 4

So, the solution is x = 4.

Step-by-Step Process

To solve any linear equation, follow these steps:

  • Step 1: Simplify both sides of the equation, if needed.
  • Step 2: Use addition or subtraction to move constants to one side of the equation.
  • Step 3: Use multiplication or division to solve for the variable.
