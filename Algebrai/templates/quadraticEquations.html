What is a Quadratic Equation?

A quadratic equation is an equation where the highest exponent of the variable (usually x) is 2. It represents a parabola when plotted on a graph.

Basic Example

Let's start with a basic quadratic equation:

               x² - 5x + 6 = 0

To solve this equation, we can factor it:

  1. Find two numbers that multiply to 6 and add to -5: (-2) and (-3)
  2. Rewrite the equation as: (x - 2)(x - 3) = 0
  3. Set each factor to 0 and solve for x: x - 2 = 0 or x - 3 = 0
  4. Solve for x: x = 2 or x = 3

So, the solutions are x = 2 and x = 3.

Step-by-Step Process

To solve any quadratic equation, follow these methods:

  • Factoring: Factor the equation and solve for the values of x.
  • Quadratic Formula: Use the formula x = [-b ± √(b² - 4ac)] / 2a.
  • Completing the Square: Rewrite the equation as a perfect square trinomial, then solve for x.

Using the Quadratic Formula

If factoring is difficult or impossible, use the quadratic formula:

               x = [-b ± √(b² - 4ac)] / 2a

Here’s an example:

               2x² - 4x - 6 = 0

Substitute the values into the formula:

  • a = 2, b = -4, c = -6
  • Discriminant: b² - 4ac = (-4)² - 4(2)(-6) = 16 + 48 = 64
  • Apply the quadratic formula: x = [4 ± √64] / 4
  • Simplify: x = (4 ± 8) / 4, giving x = 3 or x = -1

So, the solutions are x = 3 and x = -1.
