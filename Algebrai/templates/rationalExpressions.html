What is a Rational Expression?

A rational expression is a fraction where both the numerator and the denominator are polynomials. The denominator cannot be zero, and understanding how to simplify and perform operations with these expressions is key in algebra.

Simplifying Rational Expressions

To simplify a rational expression, factor both the numerator and the denominator and cancel out any common factors.

Example:

(x² - 9) / (x² - 4x + 3) = (x + 3)(x - 3) / (x - 1)(x - 3) = (x + 3) / (x - 1)

Multiplying and Dividing Rational Expressions

When multiplying rational expressions, multiply the numerators and multiply the denominators, then simplify.

Example (Multiplication):

(x / 2) * (3 / x²) = 3 / 2x

For division, multiply the first fraction by the reciprocal of the second fraction.

Example (Division):

(x / 3) ÷ (2 / x²) = (x / 3) * (x² / 2) = x³ / 6

Adding and Subtracting Rational Expressions

To add or subtract rational expressions, first find a common denominator, rewrite each expression with this denominator, then add or subtract the numerators.

Example:

(1 / (x + 1)) + (2 / (x - 1)) = (x - 1 + 2(x + 1)) / ((x + 1)(x - 1)) = (x + 1) / ((x + 1)(x - 1)) = 1 / (x - 1)

Restrictions on the Domain

When dealing with rational expressions, it’s important to state any restrictions on the domain, meaning values of the variable that would make the denominator zero.

Example:

For the expression (x + 3) / (x - 2), the value x = 2 is not allowed, because it makes the denominator zero.
